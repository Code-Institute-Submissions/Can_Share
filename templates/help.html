Need Some Help?

How to add a can?

Login then select add a beer Simply fill out all fields and click add beer done!

Is my Beer Vegan Friendly?

As a general rule if the beer is hazy (Unfined) then generally it is ok for vegan consumption, however this is not always the case, check on the bottle or can for more info, if in doubt select no.

Made a mistake?

Select the beer you added and click Edit beer Make your corrections and click done Stop and marvel at your new beer review creation raise a glass and shout Cheers!

How to add links to images an image url?

Find an image on the internet you would like to use, it's recomended to find a square image, please ensure there is no copy write and you have the owner's permission to use the image.

Desktop:

Right click on the image (On Google chrome and Opera) select copy image address (FireFox) copy image location. Then paste that link in the image URL box.

Mobile:

Find the image press and hold a small box will popup tap the image again then select copy link address, then paste in to image url field. ** Note this has only been tested on android with chrome other makes and browsers maybe different please consult your phone manufacture documentation.**

Unfortunately as yet we do not have the facility to upload images, however if the images are your own they can be uploaded to image hosing sites like https://imgur.com/ for free.
